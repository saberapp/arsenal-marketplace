# Saber Marketplace

The official Claude Code plugin marketplace for [Saber](https://saber.app) — GTM and RevOps tools that bring revenue intelligence directly into your Claude Code workflow.

## Installation

```
/plugin marketplace add saberapp/saber-marketplace
```

## Plugins

### [saber-arsenal](https://github.com/saberapp/arsenal)

GTM and RevOps skills for Claude Code — prospecting, signal tracking, ICP extraction, account scoring, outreach, and more.

```
/plugin install saber-arsenal@saber-marketplace
```
